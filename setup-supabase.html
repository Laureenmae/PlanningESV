⚙️ Configuration Supabase

ℹ️ Info: Cette page vous guide pour configurer le stockage partagé. Suivez les étapes ci-dessous.
1
Créer un compte Supabase
Allez sur supabase.com et inscrivez-vous avec votre email.
2
Créer un nouveau projet
Dans le dashboard Supabase, cliquez sur "New Project". Donnez-lui un nom (ex: "planning-2026") et choisissez une région proche de vous.
3
Exécuter le schéma SQL
Étape A: Allez dans "SQL Editor" → "New Query"
Étape B: Copiez le contenu du fichier schema.sql du repo
Étape C: Collez-le dans l'éditeur et cliquez "Run"
4
Récupérer vos clés API
Allez dans Settings → API et copiez:
  • Project URL (commence par https://)
  • anon public key
5
Mettre à jour index.html
Collez vos clés dans le fichier index.html aux lignes indiquées:
// Ligne ~45 dans index.html const SUPABASE_URL = "YOUR_URL_HERE"; const SUPABASE_ANON_KEY = "YOUR_KEY_HERE";
6
Tester la connexion
Allez sur votre site et vérifiez que le status dit "Enregistré ✓" (pas "⚠ Sauvegarde indisponible")
✓ Configuration générée! Copiez le code ci-dessous dans index.html
